evilmc: Ellipsoidal Variations Induced by Low-Mass Companions
==============================================================

Welcome to the documentation for ``evilmc``, a Python package for calculating ellipsoidal variations in planetary systems. 

Please cite the `original paper <http://adsabs.harvard.edu/abs/2012ApJ...751..112J>`_ if you make use of ``evilmc`` in your work!
